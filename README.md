# AI Stock Research Assistant
A powerful Streamlit application that leverages Google's Gemini AI to provide comprehensive stock market analysis. This assistant can answer questions about stock prices, company information, market news, financial data, and overall market trends.

## Features
```bash
1.Real-time Stock Information: Get current stock prices, market cap, and company details
2.Latest News: Fetch recent news articles related to specific stocks
3.Financial Analysis: Access key financial metrics including revenue, net income, and operating income
4.Conversational Interface: Chat with the AI assistant to get insights about the stock market
5.Dark/Light Mode: Toggle between dark and light themes for comfortable viewing
```
## Installation
### Clone this repository:
```bash
git clone https://github.com/Abhishek-DS-ML-Gupta/StockAgent.git
cd StockAgent
```
### Install the required dependencies:
```bash
pip install -r requirements.txt
```
#### Set up your Google Gemini API key:
- Get an API key from [Google AI Studio](https://aistudio.google.com/api-keys)
Replace the placeholder API key in the code with your own
Usage.
### Run the Streamlit app:
```bash
streamlit run stock.py
```
